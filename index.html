Servicios de informartica

Desarrollo Web
Backend, Frontend, Diseño, UX, usabilidad, posicionamiento, estrategia, arquitectura y muchos otros términos componen el diseño y desarrollo web moderno.

Tiendas Virtuales

CSM Wordpress

CSM Joomla
Realizamos portales completos, modificaciones, ascesoria y otros en el Content Management System de Joomla.

Apis Rest
En la actualidad no existe proyecto o aplicación que no disponga de una API REST para la creación de servicios profesionales a partir de ese software. Twitter, YouTube, los sistemas de identificación con Facebook y muchos mas, hay cientos de empresas que generan negocio gracias a REST y las APIS REST.

Base de Datos
